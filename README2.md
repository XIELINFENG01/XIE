# C语言学习（穷举法和递推法）
## 穷举法：在条件范围内对所有可能情况逐个验证；（循环或if实现）（本质是列举所有可能）
* 例子：求密码
（```）
  int main()
 {
   long m;
   for(m=67008;m<=67998;m+=10)
   {
     if(m%78==0&&m%67==0)
 	{
 	 printf("%ld",m);
	}
    }
    return 0;
 }
（```）
